# 🌍 Climate Trend Analyzer (Google Colab Version)

## 📌 Project Overview

The **Climate Trend Analyzer** is a data science project that analyzes historical climate data to identify long-term trends, detect anomalies, and predict future temperature patterns.

This project is implemented using **Google Colab**, making it easy to run without any local setup.

---

## 🎯 Objectives

* Analyze climate data (temperature, rainfall, CO₂, humidity)
* Identify long-term trends and seasonal patterns
* Detect anomalies (extreme climate events)
* Forecast future temperature using machine learning

---

## 🚀 Features

* 📊 Data visualization (trend graphs)
* 🔍 Anomaly detection
* 📈 Temperature forecasting (Linear Regression)
* ☁️ Fully executable in Google Colab (no installation required)

---
