# 🌍 Climate Trend Analyzer

## 📌 Project Overview

This project analyzes historical climate data to identify trends, anomalies, and future predictions.

## 🎯 Features

* Temperature trend analysis
* Anomaly detection
* Climate forecasting
* Interactive dashboard (Streamlit)

## 🛠 Tech Stack

* Python
* Pandas
* Matplotlib
* Scikit-learn
* Streamlit

## 📂 Project Structure

```
Climate-Trend-Analyzer/
```

## ▶️ How to Run

### 1. Install dependencies

```
pip install -r requirements.txt
```

### 2. Run main project

```
python main.py
```

### 3. Run dashboard

```
streamlit run app/app.py
```

## 📊 Outputs

* Graphs saved in outputs/
* Prediction values
* Anomaly detection results
